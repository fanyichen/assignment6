The program can be run through the command 'python assignmentment6.py' in the ass502 directory. In order to quit the program, the user can either type quit, ctrl-c, or ctrl-d. An input of any of these will immediately end the program. There are two test modules, test_interval and test_merge. They test valid creation of interval objects as well as various functions related to merging intervals. They can be run through the command 'python -m unittest discovery' in the ass502 directory. Tests can be run individually through a command such as 'python -m unittest -v test_interval'.
